C/C++万能头文件——bits/stdc++.h

本文最后更新于：2021年4月26日 晚上

#C/C万能头文件——bits/stdc.h

最近跟着打ACM的思路学算法时发现有这个好东西，感觉可以很省时间，但在 VS Code 里用发现会报错，找到编译器所在文件夹的include文件夹..\MinGW\lib\gcc\x86_64-w64-mingw32\8.1.0\include\c++\bits新建一个stdc++.h文件，再把内容复制进去即可。
